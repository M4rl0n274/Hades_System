# 🧾 Hades System - Aplicación de Facturación

Aplicación web desarrollada con Flask para la gestión de clientes, productos y facturas.
Este proyecto forma parte de un prototipo funcional orientado a la administración básica de ventas.

---

## 🚀 Tecnologías utilizadas

* Python
* Flask
* HTML5
* Bootstrap 4
* JavaScript

---

## ⚙️ Requisitos

Antes de ejecutar el proyecto asegúrate de tener instalado:

* Python 3.x
* pip (gestor de paquetes de Python)

---

## 🛠️ Instalación y ejecución

Sigue estos pasos para ejecutar el proyecto en tu entorno local:

### 1. Crear entorno virtual

```bash
py -m venv .venv
```

---

### 2. Activar entorno virtual

En Windows:

```bash
.venv/Scripts/activate
```

---

### 3. Instalar dependencias

```bash
pip install flask
```

---

### 4. Ejecutar la aplicación

```bash
flask run
```

---

## 🌐 Acceso a la aplicación

Una vez ejecutado el servidor, abre tu navegador y accede a:

```
http://127.0.0.1:5000/
```

---
